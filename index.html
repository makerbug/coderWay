概述

什么是Stream

Stream是java8提出的一种新特性,Stream流的特点.
链式调用: 可以通过一系列的方法调用来定义对流的操作,使得代码更具可读性.
惰性求值: 流上的操作不会立即执行,只有在遇到终端操作时才会触发计算.
函数式编程: 流操作使用了函数式编程的思想,可以通过Lambda表达式来定义操作.
并行处理: 可以轻松地将流操作并行化,充分利用多核处理器性能.

概述

Byte是Java八大基本数据类型之一
将字符串转换成byte[]存储需要借助一个字符集,该字符集指定了每个字符所对应的数字,字符串内的每个字符根据字符集转换成指定的数字后存入byte[].
由多个字符组成的byte[]就组成了媒体文件,图片,各种文档等等的源文件.

字符集 编码

  1. UTF-8编码
    一个英文字符等于一个字节,一个中文等于三个字节
  2. Unicode编码
    一个英文等于两个字节,一个中文等于两个字节

spring-bootstrap及spring多配置文件

发表于 2023-10-27 更新于 2023-10-30 分类于 spring

概述

bootstrap.yml是Spring Cloud Config中用于获取外部配置的特殊文件。他的加载优先级比其他配置文件（application.yml或application.properties）更高。

bootstrap.yml主要用于加载一些启动阶段需要用到的配置信息，例如连接到Spring Cloud Config Server的配置，加密/解密信息，连接到外部数据库的配置等，因为这些配置信息在应用程序启动的早期阶段就需要用到，所以放在bootstrap.yml中确保它们在其他配置之前加载。

bootstrap.yml不会被本地配置文件覆盖，他的优先级更高，确保这些关键配置的安全性和准确性。
